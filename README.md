# 🚀 Pathly

AI-powered career roadmap platform that analyzes resumes, identifies skill gaps, and generates personalized learning paths for target roles.

## 🌐 Live Demo

Website: https://pathly-ebon.vercel.app

GitHub Repository: https://github.com/mahitha-chippa4/pathly

---

## ✨ Features

* Google, GitHub & Email Authentication
* Resume Upload & Parsing
* Career Role Matching
* Skill Gap Analysis
* Readiness Score
* Personalized Career Roadmap Generation
* Dashboard & Progress Tracking
* Resume-Based Recommendations

---

## 🏗 System Architecture

User
↓
Next.js Frontend
↓
NextAuth Authentication
↓
PostgreSQL (Neon)
↓
Resume Processing API
↓
ML Matching Engine
↓
Roadmap Generator
↓
Personalized Career Roadmap
